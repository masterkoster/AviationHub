# Changelog

All notable changes to AviationHub will be documented in this file.

## [0.1.0] - 2026-06-25

### Added
- Initial desktop application
- Pilot logbook with full time breakdowns (PIC, SIC, night, instrument, cross-country)
- Interactive airport map with 20,000+ airports
- Route planner with waypoints, GPX/FPL/JSON import/export
- Weight & Balance calculator with CG visualization
- Fuel planning with price comparison
- Route weather (METAR, TAF, wind aloft)
- FAA currency tracking (local + cloud)
- Aircraft management (CRUD, N-number tracking)
- Calendar/agenda for event tracking
- Encrypted backup and restore (.ahb files)
- Local mode (offline-first, PIN-protected)
- Cloud mode (sync across devices)
- PS4-style account picker
- First-time setup wizard
- Theme support (Light, Dark, System)
- Global keyboard shortcuts (Ctrl+1-8 navigation, Ctrl+K palette)
- Command palette for quick actions
- Auto-updater with in-app update banner
- GitHub Actions CI for Windows builds

### Notes
- Web version is in development (join waitlist at aviationhub.app)
